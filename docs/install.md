# Installation Notes

## Lab Environment

This lab uses two Rocky Linux virtual machines.

| Hostname |       Role         |       OS    |
|----------|--------------------|-------------|
| server1  | Admin / Web Server | Rocky Linux |
| server2  | Client / Test Node | Rocky Linux |

## VM Plan

Recommended VM settings:

- CPU: 2 cores
- Memory: 2GB - 4GB
- Disk: 30GB - 40GB
- Network Adapter 1: NAT
- Network Adapter 2: Host-only
