# Linux Infrastructure Lab

## Project Overview ｜ 專案概述

This is a hands-on Linux infrastructure project built for my career transition from COBOL / Mainframe systems to Linux / Infrastructure / DevOps engineering.

這是一個實作型 Linux 基礎架構專案，用於展示我從 COBOL / Mainframe 領域轉型至 Linux / Infrastructure / DevOps 的能力。

---

## Background ｜ 背景說明

I previously worked in large enterprise environments with COBOL / Mainframe systems, handling system maintenance, batch processes, reverse engineering, and technical documentation.
This project represents my active transition into modern infrastructure and operations roles.

過去我曾於大型企業環境中從事 COBOL / Mainframe 系統維運、批次流程分析、逆向理解與技術文件撰寫。
此專案代表我正積極轉型至現代化基礎架構與維運工程領域。

---

## Project Goals ｜ 專案目標

- Build a small Linux lab environment with multiple virtual machines  
- Practice SSH remote administration  
- Manage users, groups, and sudo permissions  
- Configure firewall security rules  
- Deploy web services with nginx or httpd  
- Perform troubleshooting using logs and system tools  
- Document the full implementation process  

- 建立小型多主機 Linux Lab 環境  
- 練習 SSH 遠端管理  
- 使用者、群組與 sudo 權限管理  
- 防火牆安全設定  
- 部署 nginx / httpd Web 服務  
- 使用 log 與系統工具進行除錯  
- 完整記錄建置流程  

---

## Planned Lab Architecture ｜ 預計架構

- server1 : Admin / Web Server  
- server2 : Client / Test Node  
- Network : NAT + Host-only  
- OS : Rocky Linux  

- server1：管理主機 / Web Server  
- server2：測試主機  
- 網路模式：NAT + Host-only  
- 作業系統：Rocky Linux
